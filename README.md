Pose and Anatomy Detection for Artists and Creators [Python, OpenCV, MediaPipe]
Real-time human pose estimation system using camera feed to assist artists and animators.

•Utilized MediaPipe's Pose module to detect 33 human body landmarks
•Accessed and processed live video with OpenCV, displaying real-time pose overlays
•Landmarks visualized using drawing utilities for reference and posture accuracy
Ideal for digital artists, animators, and motion study creators seeking live anatomy aid
